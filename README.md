# Практика: Изучение возможностей библиотеки VPI для видеокарт NVIDIA

## Описание проекта
В рамках практики были реализованы четыре вычислительные задачи на трёх платформах: Python (CPU), VPI (GPU) и CUDA C++ (GPU).

## Содержание репозитория
| Файл | Описание |
|------|----------|
| `monte_carlo_python.py` | Метод Монте-Карло на Python |
| `monte_carlo.cu` | Метод Монте-Карло на CUDA |
| `laplace_python.py` | Уравнение Лапласа на Python |
| `laplace.cu` | Уравнение Лапласа на CUDA |
| `gauss_python.py` | Метод Гаусса на Python |
| `gauss.cu` | Метод Гаусса на CUDA |
| `image_python.py` | Обработка изображений на Python (OpenCV) |
| `vpi_image.cpp` | Обработка изображений на VPI (C++) |
| `image_cuda.cu` | Обработка изображений на CUDA (C++) |
| `Makefile` | Сборка VPI-программы |
